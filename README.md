# Student Management System 🎓

A web-based app to manage student records. Built with PHP, Javascript, MySQL, and HTML.

## 🚀 Features
- Add and delete student profiles

## 🛠️ Tech Stack
- **Backend:** PHP, MySQL
- **Frontend:** HTML, CSS, JavaScript
